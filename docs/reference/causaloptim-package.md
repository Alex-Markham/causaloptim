# An Interface to Specify Causal Graphs and Compute Bounds on Causal Effects

Specify causal graphs using a visual interactive interface and then
analyze them and compute symbolic bounds for the causal effects in terms
of the observable parameters.

## Details

Run the shiny app by `results <- specify_graph()`. See detailed
instructions in the vignette `browseVignettes("causaloptim")`.

## References

Sachs, M. C., Jonzon, G., Sjölander, A., & Gabriel, E. E. (2023). A
